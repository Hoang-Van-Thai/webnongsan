# Agricultural Products E-commerce Website

## Overview

An e-commerce website developed using ASP.NET MVC that allows users to browse, search, and purchase agricultural products online. The system also provides an administration panel for managing products, categories, orders, and customers.

## Features

* User Registration & Login
* Product Management (CRUD)
* Category Management
* Shopping Cart
* Order Management
* Customer Management
* Search Products
* Responsive User Interface

## Technologies

* C#
* ASP.NET MVC
* Entity Framework
* SQL Server
* HTML5
* CSS3
* JavaScript
* Bootstrap
* Git

## Project Structure

* Models
* Views
* Controllers
* Database
* Assets

## How to Run

1. Clone this repository.
2. Open the solution in Visual Studio.
3. Restore NuGet packages.
4. Configure SQL Server connection string.
5. Run database migration or restore the database.
6. Press **F5** to start the application.
